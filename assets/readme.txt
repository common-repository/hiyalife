Publish your post in Hiyalife.com and view your timeline in your Wordpress blog.

== Description ==

A light weight wordpress plugin that allows your WordPress Blog to connect with Hiyalife in two ways:-	Publishing posts from WordPress to Hiyalife-	Inserting Hiyalife timelines into Wordpress using short codesPublishing to HiyalifeTitle, text, and pictured included in a wordpress post will be automatically uploaded into Hiyalife Hiyalife Timeline WidgetProvides the configuration screen to obtain and insert the shortcode of a timeline widget into your posts in WordPress

== Installation ==

1. Install Hiyalife plugin either via the WordPress.org plugin directory or by uploading the files to your wordpress dashboard under plugins->Add New
2. Activate the plugin
3. After activating Hiyalife plugin, you will need to go to "Hiyalife Settings" in Wordpress "Settings Menu" to configure it.
4. If you don't have a Hiyalife account yet, you can quickly create one in www.hiyalife.com .
5. Click the "Reedem Your Keys" link and copy/paste the consumer key and consumer secret key.
6.Check the Publish by default option if you wish. If not checked you will be able to publish in a post by post basics
7. Click the "Save Changes" button and the plugin is ready to go!Now everytime you publish you will be able to send content to Hiyalife
8. If you want to use the Hiyalife widget to include timelines in your pages you will not need to redeem your keys and type them in your settings screen
